# Asset

To obtain the information of a single asset of a user by asset id, the `ASSETS:READ` permission is required.

### `GET /assets/:id`

```
$$XIN:curl$$ "https://api.mixin.one/assets/3596ab64-a575-39ad-964e-43b37f44e8cb"
```

```json
{
  "data": $$XIN:asset$$
}
```
